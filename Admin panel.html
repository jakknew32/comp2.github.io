📋 วิธีสร้าง Firebase Project
1 สร้าง Firebase Project
ไปที่ console.firebase.google.com → คลิก Add project → ตั้งชื่อ → กด Continue จนเสร็จ
2 เปิดใช้ Firestore Database
ในเมนูซ้าย → Firestore Database → Create database → เลือก Start in test mode → เลือก region ใกล้สุด (เช่น asia-southeast1) → Enable
3 คัดลอก Config
ไปที่ Project Settings (⚙️) → Your apps → คลิก </> Web → ตั้งชื่อ app → Register app → จะเห็น firebaseConfig → คัดลอกทั้งหมด
4 วาง Config ด้านล่าง
วาง firebaseConfig object (ทั้ง { }) ลงในช่องด้านล่าง
